Descripción del Proyecto
**AluraFlix** es una aplicación web interactiva inspirada en las plataformas de streaming modernas. Fue desarrollada como el Challenge final del módulo de React en Alura Latam, con el objetivo de consolidar el manejo de componentes funcionales, estados globales y flujos de datos en el Frontend.

La aplicación permite organizar contenido audiovisual por categorías (como Front End, Back End, Innovación, etc.), ofreciendo una interfaz fluida, responsiva y completamente dinámica para el usuario.

Características Principales

*   **Arquitectura basada en Componentes:** Modularización limpia utilizando React.
*   **Gestión de Estado Dinámica:** Formularios validados para la creación e incorporación de nuevas tarjetas de video a la plataforma.
*   **Categorización por Colores:** Identificación visual de cada sección para mejorar la experiencia de usuario (UX).
*   **Diseño Totalmente Responsivo:** Interfaz adaptada para una navegación fluida tanto en dispositivos móviles como en pantallas de escritorio.

Tecnologías Utilizadas
*   **React.js** (Componentes funcionales y React Hooks)
*   **JavaScript (ES6+)**
*   **CSS3 / Styled Components** (para el estilado modular y dinámico)
*   **Vercel** (para el despliegue continuo en producción)


*Desafío técnico desarrollado originalmente en el marco de la formación Front End de Alura Latam y optimizado para el portafolio profesional de Vowk Studio.*
